# タッチ50音ドリル

[自動採点付きの50音ドリル](https://marmooo.github.io/touch-50on/)です。

## Installation

install [KanjiVG/kanjivg](https://github.com/KanjiVG/kanjivg) licensed under the
[CC BY-SA 3.0](https://creativecommons.org/licenses/by/3.0/)

## Build

```
bash build.sh
```
