# タッチ50音ドリル
[自動採点付きの50音ドリル](https://marmooo.github.io/touch-50on/)です。

## Installation
1. install [KanjiVG/kanjivg](https://github.com/KanjiVG/kanjivg)
2. ```npm install```

## Build
```bash build.sh```
